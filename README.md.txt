# FC Barcelona Performance Analysis

## Project Overview
This project analyzes the performance of FC Barcelona across multiple La Liga seasons (2021–2025) using Python and data analysis techniques.

The goal is to extract insights about team performance, scoring trends, and match outcomes using real-world football data.

---

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

---

## Dataset
- Source: football-data.co.uk (La Liga data)
- Seasons analyzed: 2021–2025

---

## Objectives
- Analyze Barcelona match outcomes (Win/Loss/Draw)
- Compare home vs away performance
- Study goal-scoring patterns
- Evaluate performance trends over multiple seasons
- Identify toughest opponents

## Key Insights
- Barcelona shows stronger performance in home matches compared to away games.
- Goal scoring varies across seasons, indicating fluctuations in attacking performance.
- Win rate highlights overall consistency and competitive strength.
- Certain opponents consistently pose challenges to Barcelona.
